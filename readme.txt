Transform every WordPress gallery in a postcards system!

== Description ==

Postcards transforms every WordPress gallery within a post or a page in a postcards system
that lets the user to send a gallery image as an ecard.

Full documentation and examples can be found on
[Postcards official page](http://www.satollo.net/plugins/postcards). 

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
